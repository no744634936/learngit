1,先在github上面建立一个repository。名字为learngit


2,在桌面建立一个文件夹。 learngit


3,learngit文件夹里添加文件。

4,git add .

5,git commit -m "......."

6,git remote add origin https://github.com/no744634936/learngit.git

7,git push -u origin master(往master分支上push)

第二次往远程仓库里面push的时候就可以，这样了

git add .
git commit -m "....."
git push


8,获取更新的时候就要用 git pull origin master(获取origin 上面的master branch 的更新)
